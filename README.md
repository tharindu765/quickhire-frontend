# QuickHire Frontend

React frontend for the QuickHire freelancer marketplace platform.

## Tech Stack

- React 19
- Vite 8
- React Router 7
- Axios

## Getting Started
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deployment

Deployed on Google Cloud Run using buildpacks.
```bash
npm start
```
